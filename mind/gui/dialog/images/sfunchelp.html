How to use Function Editor.

Expressions can be written very freely in the spreadsheet.

Cell references are written with an '#' before the column and row name i.e. "#A4". Just remember that the root of the expression is where an comparision operator ('=', '>'or '<') is present.

Comments should start with an '%'. Comments can also be written with no special marker if one avoid to use '=', '>'or '<' in the text.
